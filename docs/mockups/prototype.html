3 · 设计语言与快捷键

视觉规范

  • 系统字体SF Pro / 苹方,数字列用 .monospacedDigit()
  • 配色 · 涨#27AE60(West) / #E74C3C(East)
  • 配色 · 跌#E74C3C(West) / #27AE60(East)
  • 主色#7C5CFF(强调 / Active 状态)
  • 圆角14pt Popover / 8pt 卡片 / 6pt 按钮
  • 背板Material thin + 80% 不透明

键盘快捷键

  • ⌘P显示 / 隐藏 Popover
  • ⌘R立即刷新行情
  • ⌘,打开设置
  • ⌘F在 Holdings 中搜索
  • ⌘1-4切换 Tab
  • ⌘Q退出 PanBar
