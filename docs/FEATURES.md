# PanBar 功能清单

> 按 **MoSCoW** 优先级划分功能范围。
> Must / Should / Could / Won't = 必须 / 应该 / 可以 / 暂不

---

## 🟥 Must Have (Phase 1 / MVP)

### M1. 菜单栏滚动行情
- M1.1 在 macOS 菜单栏常驻显示一个图标 + 滚动文字区
- M1.2 滚动展示用户配置的股票:`代码 价格 涨跌幅`
- M1.3 鼠标 hover 时暂停滚动
- M1.4 支持 A 股、港股、美股
- M1.5 涨跌配色方案(East / West / Monochrome 三选一)
- M1.6 自动识别交易时段(休市时减速并标注 "Closed")

### M2. 持仓管理
- M2.1 手动添加 / 编辑 / 删除持仓(股票代码、数量、成本价、币种)
- M2.2 列表展示:代码 / 名称 / 现价 / 涨跌幅 / 持仓数 / 盈亏金额
- M2.3 持仓数据本地持久化(SQLite via GRDB)

### M3. 自选股(Watchlist)
- M3.1 添加 / 删除 / 拖拽排序
- M3.2 区别于持仓:不计算盈亏,只看行情

### M4. Popover 主面板
- M4.1 点击菜单栏图标弹出,360pt 宽
- M4.2 顶部 Summary:总资产 / 今日盈亏 / 总盈亏(按本位币换算)
- M4.3 三个 Tab:Holdings / Watchlist / Indices
- M4.4 Footer:刷新按钮 + 上次更新时间 + Quit

### M5. 数据源
- M5.1 内置腾讯财经接口(`qt.gtimg.cn`)
- M5.2 内置东方财富接口(`push2.eastmoney.com`)
- M5.3 自动 fallback:腾讯失败切东财
- M5.4 请求合并 + 节流(同市场批量拉取)

### M6. 多币种本位币汇总
- M6.1 设置可选 Base Currency(CNY / USD / HKD)
- M6.2 FX 汇率源:东方财富外汇接口
- M6.3 Summary 区按本位币展示,明细保留原币种

### M7. 基础设置窗口
- M7.1 General:开机启动 / 主题 / 语言 / 本位币
- M7.2 Portfolio:增删改持仓
- M7.3 About:版本号 / GitHub / License

### M8. 系统集成
- M8.1 通过 `SMAppService` 实现开机启动
- M8.2 跟随系统亮 / 暗模式
- M8.3 国际化:简体中文 + English

## 🟧 Should Have (Phase 2)

### S1. 大盘指数 Tab
- S1.1 预置:上证、深证、创业板、恒生、纳指、标普、道指
- S1.2 用户可勾选展示哪些
- S1.3 Popover 内 Indices Tab 独立卡片展示

### S2. 价格预警(Alerts)
- S2.1 添加预警规则(标的 + 条件 + 阈值)
- S2.2 条件类型:价格 ≥ / ≤,日涨跌幅 ≥ / ≤
- S2.3 通过 `UNUserNotificationCenter` 发本地通知
- S2.4 触发后进入"已触发"态(避免重复轰炸)
- S2.5 Snooze 1h / Disable / Reset 操作

### S3. 滚动行为定制
- S3.1 滚动速度三档(慢 / 中 / 快)
- S3.2 显示模板预设(Compact / Standard / Detailed)
- S3.3 自定义模板字符串(`{symbol} {price} {change_pct}`)
- S3.4 菜单栏最多展示数量(避免占太长)

### S4. 数据导入导出
- S4.1 CSV 导入持仓 / 自选股
- S4.2 CSV 导出
- S4.3 配置文件备份 / 恢复

### S5. 右键菜单
- S5.1 菜单栏右键 → 暂停滚动 / 刷新 / 设置 / 退出
- S5.2 Popover 列表右键 → 编辑 / 删除 / 设置预警 / 浏览器打开

### S6. 智能刷新
- S6.1 监听系统休眠 / 屏幕锁定 → 暂停请求
- S6.2 监听网络状态 → 离线时灰显 + 提示
- S6.3 不同状态下不同刷新间隔(详见 PROTOTYPE.md §5)

## 🟩 Won't Have (v1 明确不做)

- ❌ **加密货币**:聚焦股票场景
- ❌ **真实下单交易**:只做行情查看,不接券商 API
- ❌ **AI 分析**:不在菜单栏小工具范畴
- ❌ **社交 / 评论**:保持工具属性
- ❌ **Windows / Linux 版本**:菜单栏交互是 macOS 独有形态
- ❌ **iOS App**:首版只做 Mac,iOS 后续按需考虑
- ❌ **图表 / K 线**:大图表用浏览器,菜单栏不适合
- ❌ **云同步用户数据**:无服务端,纯本地

## 验收标准(MVP 完成定义)

满足以下全部 Must 项后,即可标记为 v0.1.0 可发布:

- [ ] 添加至少 5 只跨市场股票,菜单栏正确滚动 30 分钟无崩溃
- [ ] 持仓盈亏计算正确(对照 Yahoo Finance 校验,误差 < 0.1%)
- [ ] 多币种持仓汇总到 CNY 正确(已知汇率手动校验)
- [ ] 数据源 fallback 生效:断开腾讯 IP 后自动切东财
- [ ] 内存占用 < 80 MB,CPU 闲时 < 1%
- [ ] 开机启动正常,深色模式视觉无 bug
- [ ] 中英文 UI 全量切换无遗漏
